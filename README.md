# helloodokai/code-ant

Homebrew tap for [code-ant](https://github.com/helloodokai/code-ant).

## Installation

```bash
brew tap helloodokai/code-ant
brew install code-ant
```

## Updating

When a new release is published:

1. Update `url` and `sha256` in `Formula/code-ant.rb`
2. Commit and push
